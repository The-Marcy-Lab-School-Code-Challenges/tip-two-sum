## index.js

Problem Description

Write a function named `findMax` that takes an array of numbers (`numbers`) as input and returns **the largest number in the array.**

```jsx
findMax([3, 7, 2, 9, 1]);
// → 9

findMax([15, 4, 22, 8]);
// → 22

findMax([-5, -2, -10, -1]);
// → -1
```
